# 🦆 Bebek Hub Online

Selamat datang di **Bebek Hub**, solusi pesan bebek goreng anti-ribet! Web ini dibuat menggunakan HTML, CSS, dan JavaScript murni dengan sistem database berbasis Cloud.

## 🚀 Fitur Unggulan
* **Menu Mewah & Responsif:** Tampilan cakep di HP maupun Laptop.
* **Real-time Database:** Menggunakan MockAPI untuk kirim & terima pesanan.
* **Admin Dashboard:** Dashboard khusus penjual untuk pantau pesanan masuk.
* **Voice Notification:** Laptop Admin bakal teriak *"Bos, ada pesanan!"* tiap ada pelanggan jajan.
* **Cloud Switch:** Tombol Buka/Tutup toko yang sinkron ke semua HP pelanggan secara otomatis.

## 🛠️ Teknologi yang Digunakan
* **Frontend:** HTML5, CSS3 (Flexbox & Grid), JavaScript (ES6+)
* **Database:** [MockAPI.io](https://mockapi.io)
* **Hosting:** GitHub Pages

## 📖 Cara Penggunaan
1.  Buka halaman **Pembeli** untuk melihat menu.
2.  Klik **Pesan Sekarang**, masukkan nama dan porsi.
3.  Buka halaman **Admin**, klik tombol **MULAI MONITOR** untuk mengaktifkan notifikasi suara.
4.  Gunakan tombol **Buka/Tutup Toko** di Admin untuk mengatur akses pelanggan.
